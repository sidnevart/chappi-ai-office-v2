# Edu Grant Scout

## Overview

Grant opportunities in HTML tables

## Output Formats (MANDATORY)

Every execution MUST produce:

1. **JSON** (`/mnt/files/research-state/...`) — structured data for pipeline
2. **HTML Report** (`/mnt/files/research-state/reports/html/...`) — human-readable
3. **S3 Upload** — public URL for sharing
4. **Telegram Notification** — with link to HTML report

## Pipeline

```
Agent Logic → JSON (state) → HTML (report) → S3 Upload → Telegram
```

## Workflow

### Step 1: Agent Logic
Execute skill-specific logic, save JSON to state directory.

### Step 2: Generate HTML Report
```python
import json

# Read JSON
data = json.load(open('research-state/...'))

# Generate HTML from JSON
html = generate_html_report(data)
open('reports/html/report.html', 'w').write(html)
```

### Step 3: Upload to MinIO
```python
import boto3
s3 = boto3.client('s3', endpoint_url='http://127.0.0.1:9000',
    aws_access_key_id='minio', aws_secret_access_key='minio123')
s3.put_object(Bucket='ai-office', Key='reports/report.html', Body=html, ContentType='text/html')
```

### Step 4: Send to Telegram
```bash
URL="http://80.74.25.43:9000/ai-office/reports/report.html"
curl -X POST "https://api.telegram.org/bot$TOKEN/sendMessage" \
  -d "chat_id=$CHAT_ID" -d "parse_mode=Markdown" \
  -d "text=📊 Edu Grant Scout Report: $URL"
```
