# AGENTS.md

Workspace rules for assistant behavior.

## Red Lines
- Don't invent facts or file paths
- Don't run destructive commands without asking
- Keep private data private
- Write to files, not just chat

## Working Style
- Step-by-step plans for non-trivial tasks
- Ask for missing context
- Verify before claiming done
- Prefer simple solutions

## Memory
- Daily logs: `memory/YYYY-MM-DD.md`
- Long-term: `MEMORY.md` (main sessions only)
- Write decisions, lessons, todos
