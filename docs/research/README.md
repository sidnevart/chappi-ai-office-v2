# Research Corpus

Здесь лежат итоговые research-артефакты, а не сырой чат.

Каждый полезный research должен заканчиваться markdown-файлом с минимумом:
- Вопрос
- Контекст
- Источники
- Вывод
- Что делать дальше
- Confidence
- Дата

Рекомендуемый путь:
- `docs/research/<topic>/<YYYY-MM-DD>-<slug>.md`

Не хранить здесь:
- секреты
- `.env` и конфиги с ключами
- сырые логи
- длинные копипасты без summary
